# MyMoney888 个人记账系统

一个功能强大、界面美观的个人记账系统，帮助您轻松管理个人财务。

## 功能特性

### 核心功能
- **快速记账**：支持支出、收入、转账三种交易类型
- **资产概览**：实时查看总资产、账户余额、收支趋势
- **账户管理**：添加、编辑、删除账户，设置初始余额
- **统计分析**：支出分类、收入分类、月度趋势、年度概览
- **信用卡管理**：信用卡账户、账单管理、消费分析
- **贷款管理**：贷款信息、还款计划、还款提醒
- **数据管理**：数据导出、数据导入、数据同步
- **用户管理**：用户注册、管理员功能、账号删除
- **年度回顾**：年度财务总结、消费分析、智能洞察

### 技术特性
- **响应式设计**：适配不同屏幕尺寸
- **本地数据存储**：使用LocalStorage存储数据
- **用户认证**：基于用户名的登录注册系统
- **数据同步**：支持与SQL数据库双向同步（v3.0特性）
- **智能冲突解决**：自动处理数据同步冲突
- **同步状态监控**：实时查看同步状态和历史记录

## 部署方法

### Docker部署
```bash
# 使用SQL数据库
cp .env.example .env
# 编辑.env文件，设置数据库连接信息
docker-compose up -d

# 使用NoSQL模式（本地存储）
docker-compose -f docker-compose-nosql.yml up -d
```

### 传统部署
1. 构建生产版本：`npm run build`
2. 将`dist`目录部署到Web服务器
3. 确保服务器支持Vue Router的history模式

- **前端框架**：Vue 3 + Vite

- **路由**：Vue Router
- **样式**：Tailwind CSS
- **数据存储**：LocalStorage + SQL数据库（可选）

## 使用指南

### 基本操作
1. **注册/登录**：创建账户或登录现有账户
2. **添加账户**：在账户管理页面添加您的银行账户、现金等
3. **开始记账**：在首页选择交易类型，填写金额、分类等信息
4. **查看报表**：在统计分析页面查看收支情况
5. **管理数据**：在数据管理页面导出或导入数据

### 高级功能
- **数据同步**：在数据管理页面设置数据库连接，实现数据同步
- **智能分析**：系统会自动分析您的消费习惯，提供理财建议
- **预算管理**：设置月度预算，控制消费支出
- **多账户管理**：支持多个账户之间的转账和余额管理

## 数据安全

- **本地存储**：默认使用LocalStorage存储数据，数据保存在本地
- **数据库加密**：使用SQL数据库时，数据传输和存储均加密
- **备份建议**：定期导出数据备份，确保数据安全

## 常见问题

### 数据存储在哪里？
默认情况下，数据存储在浏览器的LocalStorage中。您也可以在数据管理页面设置SQL数据库连接，实现数据同步。

### 如何备份数据？
在数据管理页面，点击"导出数据"按钮，将数据导出为JSON文件。

### 如何恢复数据？
在数据管理页面，点击"导入数据"按钮，选择之前导出的JSON文件。

### 忘记密码怎么办？
目前系统使用简单的用户名验证，无需密码。直接使用用户名登录即可。

### 支持多用户吗？
是的，系统支持多用户，每个用户有独立的记账数据。

# MyMoney888 个人记账系统

## 项目简介
MyMoney888 是一个功能完备的个人记账系统，支持多账套管理、收支分类、维度管理、统计分析等功能。

## 功能特性

### 核心功能
- **快速记账**：支持支出、收入、转账三种交易类型，可选择成员、商家、标签等维度
- **账户管理**：支持多种账户类型，实时更新余额
- **账套管理**：支持多账套创建、切换和管理
- **维度管理**：支持成员、商家、标签管理，以及收支分类的二级分类管理
- **统计分析**：支持按时间范围筛选，提供收支分类统计、月度趋势和年度概览
- **用户管理**：支持管理员账户和普通用户，管理员可删除账户

### 技术栈
- Vue 3 + Vite + Tailwind CSS
- Vue Router
- LocalStorage 数据持久化

## 安装和运行

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 数据存储
- 数据存储在浏览器的 LocalStorage 中
- 包括用户信息、账套、账户、交易记录、维度数据等

## 注意事项
- 本系统为前端项目，数据存储在本地浏览器中，清除浏览器数据会导致数据丢失
- 建议定期导出数据备份
